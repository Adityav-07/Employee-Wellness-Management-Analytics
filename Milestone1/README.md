# Employee Wellness Management Analytics

## Milestone 1 – User Authentication Module

### Project Objective

The objective of this milestone is to develop a secure User Authentication Module for the Employee Wellness Management Analytics (MoodMentor) application using Streamlit. The authentication system enables secure user registration, login, password recovery, email verification through OTP, JWT-based session management, and secure storage of user information using Neon PostgreSQL.

## Features Implemented

- Home Page
- User Registration (Sign Up)
- User Login
- Email Verification using OTP
- Forgot Password
- Password Reset
- JWT-based Authentication
- Secure Password Hashing using bcrypt
- Secure Session Management
- Neon PostgreSQL Database Integration
- Dashboard after Successful Login

## Google Colab Setup Instructions

1. Open `Authentication.ipynb` in Google Colaboratory.
2. Install all the required Python libraries.
3. Configure the required environment variables:
   - Neon PostgreSQL Database Credentials
   - JWT Secret Key
   - Gmail SMTP Email
   - Gmail App Password
   - ngrok Authentication Token
4. Run all notebook cells sequentially.
5. Launch the Streamlit application.
6. Open the generated ngrok URL in your browser.
